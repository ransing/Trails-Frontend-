

TrailBlzr is an application intended to inspire meetups out in the nature.


[Demo](https://youtu.be/j_NW0a5vQIg)

![](/public/ezgif.com-crop.gif)

- FrontEnd: 

    - Javascript-React 
     -   Auth- JWT
      -  WebSockets 
      -  Trails- API
      -  Maps- GoogleMaps API
      -  Weather- DarkSky API
      - NPM Packages 
        - DateTime: 
            - Moment.js
            - DatePicker
        - FullCalendar
        - react-alerts 
        - Flippy
        - Scroll-up,  
       - Data- 
            -   Chart.js 
            - VictoryJS
        - Interface 
            - BootStrap 
            - PureCSS 

- BackEnd:

    - Ruby- Rails 
    - B.crypt
    - Active Model Serializer 
    - RestClient (API)
    - WebSockets 
    -  Aliases/Self-Referentials

 * Installation 
    * npm install
    * npm start
